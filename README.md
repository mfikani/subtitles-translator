# Subtitles Translator

A Chrome extension that translates YouTube subtitles in real time into a language chosen by the user.

## Features (planned)
- Detect YouTube subtitles
- Translate subtitles in real time
- Support multiple languages
- Overlay translated subtitles on video

## Tech Stack
- JavaScript
- Chrome Extensions (Manifest v3)
